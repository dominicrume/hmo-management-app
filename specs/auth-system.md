# Authentication System

## Features
- User signup
- User login
- Forgot password
- Google authentication
- Wallet authentication

## Validation Rules
- Email required
- Password minimum 8 characters
- Secure password hashing

## User Flow
1. User signs up
2. Verification email sent
3. User verifies account
4. User accesses dashboard

## Security
- JWT authentication
- Session management
- Protected routes
