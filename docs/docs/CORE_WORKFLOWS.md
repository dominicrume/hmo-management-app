# Core Product Workflows

These workflows MUST NEVER break during refactoring or hardening.

## AI Brain Workflow
- Chat button must remain visible
- Suggested customer questions must remain visible
- AI conversation flow must remain functional

## Forms Workflow
- Save button must persist data correctly
- Save must not redirect incorrectly
- Form state must remain preserved

## Dashboard Workflow
- Dashboard navigation must remain stable
- User sessions must persist

## Authentication Workflow
- Login must remain functional
- Sessions must not randomly expire
