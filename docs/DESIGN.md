# Design System

## Colors (Strict HEX matching required)
- Primary: #0F1C2E (Navy) — Used for sidebars, main headers, primary buttons.
- Accent: #E8A84C (Amber) — Used for CTAs, active borders, status highlights, notifications.
- Canvas: #F8F4EF (Warm Cream) — Used for the main application background.
- Surface: #FFFFFF (White) — Used for form cards, panels, floating modals.

## Typography
- UI Font: **Sora** — Used across all headers, buttons, and navigation elements.
- Data Font: **JetBrains Mono** — Used for data display (IDs, timestamps, financial figures).

## Layout Structure (Three-Panel)
1. **Sidebar:** 240px Navy, persistent left navigation.
2. **Main Workspace:** `flex: 1` Cream background, scrolling form view.
3. **Forms Panel:** 280px White, fixed right sidebar.
4. Topbar: 56px fixed header.
5. Bottom Action Bar: 52px fixed save controls.
