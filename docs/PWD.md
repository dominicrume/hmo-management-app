# Product Working Document (PWD)

## App Overview
Matty's Place is an HMO tenant management system built for General Matlub, a Birmingham-based supported housing operator. 

## Core Features
- **Sidebar Tenant List:** Clicking loads a tenant's complete record.
- **Personal Details Form:** Form 3 of the 53 Ash Shahada pages.
- **Sessions Tracking:** Daily/Weekly/Monthly AI-assisted support sessions.
- **Ledger:** Service Charge tracking (paid/unpaid, auto-balance).
- **Intake Checklist:** On arrival to 3-day tasks.
- **Forms Panel:** Access to 7 form cards with progress badges.
- **Multi-brand:** Letterhead switching (Matty's, Ash Shahada, Reliance).
- **Compliance:** Audit trails and SHA-256 blockchain stamps on every save.
- **Print Active Form:** Export specific views easily for council submission.
